## 플러그인 범위

입력:

- 고객 질문, 상담 초안, 매수·매도 불안 상황
- 선택적으로 소수점 거래, 주문 정정·취소, 체결 지연, 주문장애, 공지 또는 FAQ 맥락

출력:

- `response-draft.md`: 상담사 또는 컴플라이언스 담당자가 검토 후 사용하는 답변 초안, 공식 URL 각주, 필수 유의 문구, 사람 검토 필수 문구
- `checklist.md`: 목적·기간·손실 감내도·주문 방식 이해·장애 증빙 여부를 묻는 확인 질문과 다음 행동
- `compliance-report.md`: 종목·가격·타이밍 추천성, 보상 단정, 원금·수익 보장, 불확실한 단정 표현 검사
- `escalation.md`: 주문장애·보상, 투자자문 요구, 공식 근거 부족 상황의 이관 판단
- `case-log.csv`: 날짜, 문의유형, 위험플래그, 사용근거URL, 처리결과 원장

## 판단 규칙

- 특정 종목, 가격, 목표가, 매수·매도 타이밍을 추천하지 않는다.
- 감정적 안심과 사실 설명을 분리한다.
- 카카오페이증권 공식 페이지, 카카오페이 고객센터, 금융위원회·금융투자협회 자료, 공개 공지와 FAQ를 우선한다.
- 최신 공지·약관·설명서 확인이 필요한 항목은 확인일과 재확인 URL을 출력한다.
- 추천 요청은 거절하고 의사결정 체크리스트와 위험 질문으로 전환한다.
- 장애·보상 문의는 공식 절차와 필요 증빙만 안내하고 보상 가능 여부를 단정하지 않는다.

## 말하면 안 되는 것

- 투자 조언, 수익 보장, 원금 보장, 목표가, 특정 종목 매수·매도 추천
- 사적 계좌·보유 종목 데이터의 해석 또는 요구
- 보상 가능성, 체결 가능성, 규제 결론의 단정
- 플러그인이 공식 고객센터나 등록 투자자문업자를 대체한다는 표현

## 접근성 메모

- 제출 폼 문항 ② URL 후보는 로그인 없이 일반 웹 fetch로 본문이 확인된 HTML 출처를 우선한다.
- 카카오페이 고객센터 FAQ는 로그인 없이 접근 가능하지만 Next.js 데이터가 스크립트에 내장되어 일반 line viewer에서는 빈 본문처럼 보일 수 있다. `curl` 원문에서는 FAQ 질문·답변을 확인했다.
- 카카오페이증권 PDF 설명서와 투자권유준칙 PDF는 다운로드는 가능하나, generic web fetch에서 Unicode decoding error 또는 한글 텍스트 추출 깨짐이 있어 문항 ② 최종 URL 후보에서는 제외하고 반증·보조 근거로만 둔다.
