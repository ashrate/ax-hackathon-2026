# 확인 체크리스트

## 케이스 요약

| 항목 | 내용 |
| --- | --- |
| 케이스 | case-1 |
| 고객 문의 | "소수점 주문을 넣었는데 왜 바로 체결이 안 되나요? 손해 보는 것 아닌가요?" |
| 문의유형 | 소수점 주문·체결 지연 |
| 핵심 불안 | 즉시 체결되지 않아 손실이 발생했는지 걱정 |
| 우선 처리 | 소수점매매 체결 구조 설명 후 주문 상태 확인 |

## 고객에게 물어볼 확인 질문

| 질문 | 이유 | 필수 여부 |
| --- | --- | --- |
| 주문한 상품이 해외주식 소수점매매가 맞나요? | 일반 1주 단위 주문과 처리 방식이 다를 수 있음 | 필수 |
| 주문을 넣은 날짜와 대략적인 시간대가 언제인가요? | 주문 접수·실행 가능 시간 확인 필요 | 필수 |
| 주문 상태가 접수, 실행대기, 체결, 취소 중 어디로 표시되나요? | 단순 지연인지 처리 완료인지 구분 | 필수 |
| 주문 취소를 원하시나요, 아니면 체결 예상 방식을 알고 싶으신가요? | 다음 행동이 취소 안내인지 설명인지 결정 | 필수 |
| 해당 주문이 투자 목적과 기간에 맞는지 다시 확인하셨나요? | 불안이 투자 판단 문제로 이어질 수 있음 | 권장 |
| 가격 변동 시 감내 가능한 손실 범위를 정해두셨나요? | 손실 불안 완화와 자기 결정 지원 | 권장 |
| 소수점 주문이 실시간 체결 방식이 아닐 수 있다는 점을 알고 계셨나요? | 주문 방식 이해 부족 여부 확인 | 필수 |

## 다음 행동 단계

1. 고객에게 개인정보나 주문번호 전체값을 공개 채널에 쓰지 말고 앱 또는 인증된 고객센터에서 주문 상태를 확인하도록 안내한다.
2. 소수점매매 FAQ의 실시간 체결 제한과 주문 처리 구조를 쉬운 말로 설명한다.
3. 최신 해외주식 소수점 거래설명서와 공지 확인이 필요한 경우 공식 공지 URL을 함께 제공한다.
4. 고객이 "손해 보상" 또는 "장애"를 주장하면 주문장애 기준 충족 여부 확인 단계로 넘긴다.
5. 고객이 특정 종목 매수·매도 판단을 묻기 시작하면 투자 추천 없이 목적, 기간, 손실 감내도 확인 질문으로 전환한다.
