# 검토용 초안 — 발송 전 상담사·컴플라이언스 확인 필수

> 아래 문안은 상담사 또는 컴플라이언스 담당자가 검토 후 사용하는 답변 초안입니다. 고객 발송 전 회사 기준, 고객별 사실관계, 최신 공지 확인이 필요합니다.

안녕하세요, 카카오페이증권입니다.

아침에 앱 접속이 되지 않아 주문을 못 하셨다면 많이 답답하셨을 것 같습니다. 먼저 보상 가능 여부는 이 답변만으로 확정할 수 없고, 공식 접수 후 주문장애 기준에 맞는지 확인이 필요합니다.

카카오페이증권의 주문장애 기준은 회사의 주문 관련 시스템 장애로 MTS 주문뿐 아니라 고객센터와 ARS를 통한 비상주문 등 어떤 방법으로도 주문이 불가능한 경우를 말합니다.[^1] 장애 상황에서는 화면 캡처나 동영상 등 자료를 남기고, 가능한 경우 고객센터 또는 ARS 비상주문을 접수해야 합니다. 비상주문 시도를 하지 않은 경우 보상 대상에서 제외될 수 있다는 안내도 있습니다.[^1]

따라서 아래 내용을 인증된 고객센터 채널로 알려 주세요. 공개 게시글이나 일반 채팅에는 계좌번호, 주문번호 전체값, 개인정보를 남기지 말아 주세요.

1. 앱이 열리지 않은 날짜와 대략적인 시간대
2. 사용 채널: 앱, 고객센터, ARS 중 어떤 방법을 시도했는지
3. 오류 화면 캡처 또는 동영상 보유 여부
4. 비상주문을 시도했는지와 연결 가능 여부
5. 매수/매도 중 어떤 주문을 하려 했는지

보상 여부는 공식 접수 후 기준 충족 여부를 확인해야 하며, 이 답변만으로 보상을 확정하지 않습니다. 주식 투자는 원금 손실 가능성이 있고, 주문 가능 여부와 별개로 시장가격은 변동될 수 있습니다.
