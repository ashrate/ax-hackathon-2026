# 카카오페이증권 제출용 플러그인

> AX 인재전쟁 2026 예선 제출 후보: KakaoPay Securities Investor Reassurance Copilot

## 1. 플러그인 개요

이 플러그인은 초보 투자자의 “사야 하나/팔아야 하나” 불안을 투자 추천 없이 분해하고, 공식 근거로 다음 확인 행동과 안전한 상담 답변 초안을 만드는 Codex 플러그인입니다.

사용자는 매수·매도 판단에 불안해하거나 주문 구조를 이해하지 못하는 투자자입니다. 고객지원 담당자는 이런 질문에 답할 때 특정 종목·가격·타이밍 추천처럼 보이지 않으면서도 공식 FAQ와 공지에 근거한 다음 행동을 안내해야 합니다.

## 3. 작동 방식

입력:

- 투자자 질문, 상담 초안, 매수·매도 불안 상황
- 선택 입력: 소수점 거래, 주문 정정/취소, 배당, 결제, 공지, FAQ URL 등 거래 구조 맥락

절차:

1. 질문을 투자 추천 요청, 거래 구조 이해, 손실 불안, 상담 품질 점검으로 분류합니다.
2. 특정 종목, 가격, 타이밍, 매수·매도 지시를 요구하면 추천을 거절하고 체크리스트형 사고 과정으로 전환합니다.
3. 목적, 기간, 손실 감내도, 보유 여부, 이미 확인한 근거, 주문 방식 이해 여부를 묻습니다.
4. 공식 FAQ, 공지, 금융당국 자료에서 확인 가능한 사실과 확인 불가능한 판단을 분리합니다.
5. 고객 답변 초안에서 수익 보장, 목표가, 매수/매도 지시, 보상 단정 표현을 검사합니다.
6. 공식 근거가 부족하거나 최신성이 의심되면 공식 고객센터 확인 또는 상담사 이관으로 넘깁니다.

출력:

- 질문 유형 분류
- 투자 추천 금지 여부 판단
- 명확화 질문
- 공식 근거 체크리스트
- 고객 답변 초안
- 금지 표현 QA
- 다음 확인 행동과 이관 기준

## 4. 폴백과 제한

- 투자 조언, 수익 보장, 목표가, 특정 종목 매수·매도 추천을 하지 않습니다.
- 개인 계좌, 보유 종목, 주문 내역 같은 비공개 데이터를 입력하지 않습니다.
- 공식 출처 없이 보상, 체결, 규제 결론을 단정하지 않습니다.
- 이 플러그인이 공식 고객센터나 자격 있는 금융 전문가를 대체한다고 표현하지 않습니다.

## 5. 설치·실행 방법

```bash
python tools/make_submission.py kakaopay-securities
```

생성된 `dist/kakaopay-securities/submission.zip` 안의 `src/`가 플러그인 루트입니다. Codex에서 플러그인을 로드한 뒤 다음처럼 호출합니다.

```text
고객이 "이 주식을 지금 사야 할까요?"라고 물었을 때 투자 추천 없이 불안을 분해하고 공식 근거 기반 답변 초안을 만들어줘.
```

## 6. 검증 방법

제출 전 확인할 항목:

1. `src/.codex-plugin/plugin.json`이 validator를 통과한다.
2. `src/skills/main/SKILL.md`가 비추천형 투자 불안 분해 절차를 담고 있다.
3. `src/references/evidence.md`에 공개 근거와 금지 주장이 정리되어 있다.
4. 정상 입력으로 소수점 거래 문의를 넣으면 공식 FAQ/공지 확인 체크리스트와 고객 답변 초안이 나온다.
5. 예외 입력으로 매수 추천 요구나 추천성 상담 초안을 넣으면 추천을 거절하고 금지 표현을 표시한다.
6. `logs/`에는 AI와 나눈 전체 대화 로그 원본이 포함되어 있다.
