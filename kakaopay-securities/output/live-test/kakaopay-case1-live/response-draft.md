# 검토용 초안 — 발송 전 상담사·컴플라이언스 확인 필수

> 아래 문안은 상담사 또는 컴플라이언스 담당자가 검토 후 사용하는 답변 초안입니다. 고객 발송 전 회사 기준, 고객별 주문 상태, 최신 FAQ·공지 확인이 필요합니다. 고객 발송 전 상담사/컴플라이언스 확인이 필요합니다.

안녕하세요, 카카오페이증권입니다.

소수점 주문이 바로 체결되지 않아 걱정되셨을 것으로 보입니다. 소수점매매는 일반 실시간 주문과 다르게 주문 접수, 거래 가능 시간, 체결·배분 방식에 따라 바로 체결되지 않을 수 있습니다. 현재 주문이 정상 접수 상태인지, 체결 대기인지, 취소 가능 상태인지는 고객님의 인증된 거래 화면 또는 고객센터 확인이 필요합니다.[^1]

문의하신 내용만으로 실제 손익, 체결 시점, 체결 가격, 보상 여부를 판단하거나 보장할 수는 없습니다. 해외주식 소수점 거래 조건과 설명서는 변경될 수 있으므로 최신 공지와 FAQ 기준을 함께 확인해 주세요.[^2]

상담사는 고객님의 주문 접수 시각, 주문 유형, 주문 상태, 안내 화면을 확인한 뒤 공식 기준에 맞춰 다시 안내드리겠습니다. 확인 과정에서는 계좌번호, 주문번호 전체값 같은 개인정보를 공개 문서나 일반 대화창에 남기지 말고 인증된 고객센터 채널을 이용해 주세요.

필수 유의 문구: 주식 투자는 원금 손실 가능성이 있습니다. 특정 종목의 매수·매도 판단, 수익, 손실 회피, 체결 시점 또는 체결 가격을 확정적으로 안내할 수 없습니다. 거래 조건과 공지는 변경될 수 있어 공식 FAQ·공지의 최신 내용을 함께 확인해야 합니다.
