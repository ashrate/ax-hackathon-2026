# 컴플라이언스 자체 점검

## 검사 대상

고객은 소수점 주문이 즉시 체결되지 않는 이유와 손해 여부를 문의했다. 답변 초안은 소수점 주문의 처리 구조, 고객별 주문 상태 확인 필요, 최신 FAQ·공지 확인, 투자·보상·체결 관련 단정 금지를 안내한다.

## 금지 표현 플래그

| 검사 항목 | 판정 | 근거 | 수정 제안 |
| --- | --- | --- | --- |
| 종목·가격·타이밍 추천성 표현 | 발견 안 됨 | 특정 종목, 매수·매도 시점, 목표 가격을 제시하지 않음 | 현재처럼 주문 구조와 확인 절차만 설명 |
| 보상 단정 표현 | 발견 안 됨 | 단순 체결 지연 문의이며 보상 가능 여부를 판단하지 않음 | 장애·보상 주장 발생 시 주문장애 기준 절차로 이관 |
| 원금·수익 보장성 표현 | 발견 안 됨 | 손실 여부를 단정하지 않고 원금 손실 가능성을 안내 | "손해가 아니다", "수익이 보장된다" 같은 표현 금지 유지 |
| 불확실한 사항을 확실하다고 오인하게 하는 표현 | 주의 | 고객별 주문 상태와 최신 공지는 아직 확인 전 | "인증된 채널 확인 필요", "최신 FAQ·공지 확인 필요" 문구 유지 |
| 고객 개인정보 또는 비공개 계좌정보 요구 표현 | 발견 안 됨 | 계좌번호, 주문번호 전체값을 공개 문서에 요구하지 않음 | 필요한 상세 내역은 인증된 고객센터 채널에서만 확인 |

## 최종 판정

검토 후 사용 가능 후보. 다만 고객별 주문 상태와 최신 공지 확인 전에는 고객 발송 금지이며, 상담사/컴플라이언스 확인 후 사용해야 한다.
