# 이관 판단

## 라우팅 결과

| 구분 | 판단 |
| --- | --- |
| 1차 처리 | 소수점 주문·체결 지연 구조 설명 및 고객별 주문 상태 확인 |
| 즉시 이관 여부 | 현재 입력만으로는 즉시 이관 아님 |
| 이관 유보 사유 | 장애 증빙, 보상 요구, 투자자문 요구가 명확하지 않음 |
| 필수 확인 | 주문 접수 시각, 주문 상태, 최신 FAQ·공지, 앱 장애 또는 오류 화면 여부 |
| 사람 검토 | 고객 발송 전 상담사/컴플라이언스 확인 필요 |

## 이관 조건

| 조건 | 이관처 | 근거 |
| --- | --- | --- |
| 소수점 주문 처리 방식 문의로 확인되는 경우 | 일반 상담 처리 | 카카오페이 고객센터 소수점매매 FAQ[^1] |
| 해외주식 소수점 거래 조건 또는 설명서 최신성 확인이 필요한 경우 | 상품/업무 담당 확인 | 카카오페이증권 해외주식 소수점 거래설명서 개정 공지[^2] |
| 앱 장애, 주문 불가, 보상 요구, 오류 화면 증빙이 있는 경우 | 주문장애·보상 기준 담당 또는 고객센터 절차 확인 | 카카오페이증권 주문장애 대처방법 및 보상기준[^3] |
| 고객이 보유·매도 판단, 손실 회피 방법, 수익 가능성을 묻는 경우 | 투자자문성 답변 금지. 객관 정보 제공 또는 등록 투자자문업자 안내 | 금융위원회 투자자문 유의 안내[^4] |
| 공식 근거로 확인되지 않는 세부 운영 기준을 묻는 경우 | 공식 확인 필요로 표시 후 담당 부서 확인 | 금융상품 설명의무 가이드라인[^5] |

## 상담사 처리 문구

현재 문의는 소수점 주문의 체결 지연과 손실 불안이 결합된 케이스입니다. 먼저 고객별 주문 상태를 인증된 채널에서 확인하고, 소수점매매 FAQ와 최신 공지를 기준으로 주문 구조를 설명해 주세요. 손익, 체결 시점, 체결 가격, 보상 여부는 단정하지 않습니다. 앱 장애나 주문 불가 증빙이 확인되면 주문장애 기준 절차로 이관하고, 투자 판단 요구로 확대되면 특정 종목·가격·시점 답변을 하지 말고 객관 정보 안내로 전환합니다.
