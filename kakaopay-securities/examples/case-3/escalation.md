# 이관 판단

## 라우팅 결과

| 구분 | 판단 |
| --- | --- |
| 기본 라우팅 | 투자자문 요구로 직접 답변 불가 |
| 이관처 | 등록 투자자문업자 안내 또는 객관 정보 제공 응대 |
| 이관 사유 | 특정 종목을 지금 팔지 여부는 개별 매매 판단에 해당 |
| 고객 안내 톤 | 불안 공감 + 추천 거절 + 체크리스트 제공 |

## 이관 조건

| 조건 | 이관처 | 근거 |
| --- | --- | --- |
| 고객이 특정 종목의 매도·보유·목표가 판단을 요구 | 등록 투자자문업자 안내 | 1:1 투자자문은 금융위원회에 정식 등록된 투자자문업자만 가능하다는 안내가 있음[^1] |
| 고객이 일반적인 투자 절차나 주문 방식만 원한다고 정정 | 1차 상담사 객관 정보 제공 | 표준투자권유준칙은 투자권유를 원하지 않는 투자자에게 객관적 정보만 제공하도록 함[^2] |
| 고객이 원금 회복 또는 손실 보전을 기대 | 컴플라이언스 주의 후 객관 안내 | 금융위원회는 손실보전·이익보장 문구와 원금·고수익 보장 광고에 주의해야 한다고 안내[^1] |
| 불법 리딩방이나 미등록 자문 피해를 언급 | 금융투자협회 불법 금융투자업 신고 안내 | 금융투자협회 투자자지원센터는 제도권 금융투자회사 조회와 불법 금융투자업체 신고 경로를 안내[^3] |

## 상담사 처리 문구

이 건은 특정 종목의 매도 타이밍을 묻는 투자자문 요청이므로 매수·매도 의견을 제시하지 않습니다. 고객에게 목적, 기간, 손실 감내도, 확인한 객관 정보를 점검하도록 안내하고, 개인 맞춤 1:1 자문이 필요하면 정식 등록 투자자문업자 여부를 확인하도록 안내합니다.
