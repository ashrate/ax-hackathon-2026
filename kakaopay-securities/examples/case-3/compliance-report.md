# 컴플라이언스 자체 점검

## 검사 대상

- 고객 문의: 특정 주식을 지금 팔아야 하는지 질문
- 답변 초안: 매도·보유 추천 거절, 목적·기간·손실 감내도 체크리스트, 등록 투자자문업자 안내

## 금지 표현 플래그

| 검사 항목 | 판정 | 근거 | 수정 제안 |
| --- | --- | --- | --- |
| 종목·가격·타이밍 추천성 표현 | FLAGGED IN USER REQUEST / REMOVED IN DRAFT | 고객 원문은 "지금 팔아야 할까요"로 매도 타이밍 판단을 요구함. 금융위원회는 1:1 투자자문은 정식 등록 투자자문업자만 가능하다고 안내한다.[^1] | "지금 파세요" 대신 "목적, 기간, 손실 감내도, 객관 정보를 확인해 스스로 결정하세요"로 전환. |
| 보상 단정 표현 | PASS | 보상 문의가 아님 | 해당 없음. |
| 원금·수익 보장성 표현 | PASS | 금융위원회는 원금이나 고수익 보장 광고에 현혹되지 말라고 안내한다.[^1] | "손실은 회복됩니다" 또는 "더 떨어지지 않습니다"를 쓰지 않는다. |
| 불확실한 사항의 단정 | PASS | 금융투자협회 표준내부통제기준은 불확실한 사항에 대한 단정적 판단 제공을 금지한다.[^2] | 향후 가격 방향을 예측하지 않는다. |
| 개인정보 또는 비공개 계좌정보 요구 | PASS | 답변 초안은 보유 수량, 계좌, 주문번호를 요구하지 않음 | 개인 맞춤 자문 필요 시 등록 자문업자 확인으로 안내. |

## 최종 판정

수정 후 발송. 고객 원문은 투자자문 요구이므로 직접 답하지 않고, 현재 답변 초안처럼 추천 거절과 의사결정 체크리스트, 등록 투자자문업자 안내로 바꿔 발송한다.
