# 컴플라이언스 자체 점검

## 검사 대상

- 고객 문의: 앱 접속 불가로 주문하지 못했고 보상을 요구
- 답변 초안: 주문장애 기준, 증빙, 비상주문 시도, 보상 단정 금지 안내

## 금지 표현 플래그

| 검사 항목 | 판정 | 근거 | 수정 제안 |
| --- | --- | --- | --- |
| 종목·가격·타이밍 추천성 표현 | PASS | 답변 초안은 주문장애 절차만 안내하고 매수·매도 판단을 하지 않음 | 투자 판단 질문이 이어지면 체크리스트로 전환한다. |
| 보상 단정 표현 | PASS | 카카오페이증권 보상기준은 주문장애 정의, 비상주문 시도, 신청기한, 제외 기준을 둔다.[^1] | "보상됩니다" 대신 "공식 접수 후 기준 충족 여부 확인이 필요합니다"로 유지한다. |
| 원금·수익 보장성 표현 | PASS | 금융위원회는 손실보전·이익보장 문구와 원금·고수익 보장 광고에 주의해야 한다고 안내한다.[^2] | "손실은 모두 보전됩니다" 같은 표현을 쓰지 않는다. |
| 불확실한 사항의 단정 | PASS | 금융투자협회 표준내부통제기준은 불확실한 사항에 대한 단정적 판단 제공을 금지한다.[^3] | 장애 여부와 보상 여부를 사실 확인 전 확정하지 않는다. |
| 개인정보 또는 비공개 계좌정보 요구 | PASS | 답변 초안은 공개 채널 입력 금지와 인증 채널 확인을 안내함 | 주문가격·수량 등은 인증된 고객센터 채널에서만 확인한다. |

## 최종 판정

수정 후 발송이 아니라 현재 문안으로 발송 가능. 단, 고객의 실제 보상 접수는 상담사가 아니라 주문장애 기준 담당 절차로 이관해야 한다.
