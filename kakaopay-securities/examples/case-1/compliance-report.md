# 컴플라이언스 자체 점검

## 검사 대상

- 고객 문의: 소수점 주문 체결 지연과 손실 우려
- 답변 초안: 소수점매매 처리 구조, 최신 공지 확인, 손실·추천 유의 문구

## 금지 표현 플래그

| 검사 항목 | 판정 | 근거 | 수정 제안 |
| --- | --- | --- | --- |
| 종목·가격·타이밍 추천성 표현 | PASS | 금융투자협회 표준투자권유준칙은 투자권유를 원하지 않는 투자자에게 객관적 정보만 제공하도록 한다.[^1] | 현재 초안처럼 주문 방식 설명으로 한정한다. |
| 보상 단정 표현 | PASS | 카카오페이증권 주문장애 기준은 보상 가능 조건과 제외 기준을 별도로 둔다.[^2] | "손해가 보상됩니다" 대신 "장애·보상 주장은 기준 충족 여부 확인이 필요합니다"로 쓴다. |
| 원금·수익 보장성 표현 | PASS | 금융위원회는 손실보전·이익보장 문구와 원금·고수익 보장 광고에 주의해야 한다고 안내한다.[^3] | "손해가 아닙니다" 대신 "체결 지연만으로 손실 확정 여부를 단정할 수 없습니다"로 쓴다. |
| 불확실한 사항의 단정 | PASS | 금융투자협회 표준내부통제기준은 불확실한 사항을 확실하다고 오인하게 하는 내용을 금지한다.[^4] | 시장가격 변동 가능성과 최신 공지 확인 필요성을 함께 둔다. |
| 개인정보 또는 비공개 계좌정보 요구 | PASS | 공개 답변에는 주문번호 전체값이나 계좌정보 요청이 없음 | 필요한 경우 인증된 고객센터 채널로만 안내한다. |

## 최종 판정

발송 가능. 다만 고객이 장애·보상 주장으로 전환하면 `escalation.md` 기준에 따라 주문장애 절차로 이관해야 한다.
