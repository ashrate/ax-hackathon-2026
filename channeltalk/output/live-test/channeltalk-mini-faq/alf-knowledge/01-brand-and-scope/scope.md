# live-test-mini-faq 범위

## ALF 답변 범위

- 이 문서는 사용자가 붙여넣은 공개 FAQ 텍스트만 근거로 한 ALF 지식 초안이다.
- ALF는 배송 기간, 단순 변심 교환/반품 신청 기한, 교환/반품 제한 조건, 주문 취소 시 쿠폰 복구 조건만 답한다.

## 확정 지식

- 브랜드명은 `live-test-mini-faq`로 둔다.
- 공개 URL은 제공되지 않았고, 출처는 사용자 제공 공개 FAQ 텍스트다.

## 답변 시 주의

- 주문별 실제 배송 상태, 회수 접수 여부, 환불 금액, 쿠폰 재발급 처리 여부는 공개 FAQ만으로 확정하지 않는다.
- 사용 흔적과 포장 훼손 여부는 고객 설명만으로 판정하지 않고 상담원 확인이 필요하다고 안내한다.

## 상담원 확인 필요

- 주문번호, 결제 상태, 배송 추적, 반품 접수, 쿠폰 만료 시점 확인이 필요한 질문
- 상품 하자, 오배송, 파손, 사용 흔적 판단처럼 분쟁 가능성이 있는 질문
