# 주문 취소 시 쿠폰 복구

## ALF 답변 범위

- 주문 취소 시 쿠폰 복구 가능 조건을 안내한다.
- 쿠폰의 실제 만료일, 복구 처리 여부, 계정별 쿠폰 상태는 확정하지 않는다.

## 확정 지식

- 쿠폰은 주문 취소 시 유효기간이 남아 있으면 복구될 수 있다.
- 만료된 쿠폰은 복구되지 않는다.

## 답변 시 주의

- "복구될 수 있다"는 조건부 표현이므로 반드시 유효기간 확인이 필요하다고 안내한다.
- 쿠폰 복구가 즉시 되는지, 일부 쿠폰이 제외되는지, 취소 유형별 차이는 제공된 FAQ에 없다.

## 상담원 확인 필요

- 고객이 특정 쿠폰의 만료일 또는 복구 여부를 묻는 경우
- 취소 완료 후 쿠폰이 보이지 않는다는 문의
- 쿠폰 정책 예외, 이벤트 쿠폰, 부분 취소 관련 문의
