# 교환 및 반품 안내

## ALF 답변 범위

- 단순 변심 교환/반품 신청 기한과 제한 조건을 안내한다.
- 환불 처리 기간, 회수 택배비, 상품 하자 기준은 제공된 FAQ에 없으므로 확정하지 않는다.

## 확정 지식

- 단순 변심 교환/반품은 상품 수령 후 7일 이내 신청해야 한다.
- 사용 흔적이 있거나 포장이 훼손된 상품은 교환/반품이 어렵다.

## 답변 시 주의

- "어렵습니다"는 불가 단정이 아니라 상담원 확인이 필요한 제한 조건으로 안내한다.
- 사용 흔적, 포장 훼손, 상품 하자는 고객 설명만으로 ALF가 최종 판정하지 않는다.

## 상담원 확인 필요

- 상품 하자, 오배송, 파손, 구성품 누락
- 수령 후 7일 경과 여부가 애매한 경우
- 사용 흔적이나 포장 훼손 여부를 판단해야 하는 경우
