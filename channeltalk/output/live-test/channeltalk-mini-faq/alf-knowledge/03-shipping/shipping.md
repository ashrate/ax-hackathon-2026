# 배송 안내

## ALF 답변 범위

- 결제 완료 후 일반적인 배송 소요 기간을 안내한다.
- 실제 주문의 배송 상태나 도착 예정일은 단정하지 않는다.

## 확정 지식

- 배송은 결제 완료 후 영업일 기준 2~3일 걸린다.
- 기준 시점은 주문 접수일이 아니라 결제 완료 시점이다.

## 답변 시 주의

- 휴일, 택배사 사정, 재고 부족, 도서산간 배송비, 해외배송 가능 여부는 제공된 FAQ에 없다.
- 고객이 특정 주문의 도착일을 물으면 주문 정보 확인이 필요하므로 상담원 확인 또는 배송 조회 안내로 처리한다.

## 상담원 확인 필요

- "내 주문 언제 와요", "오늘 출고됐나요"처럼 개별 주문 상태가 필요한 질문
- 배송 지연, 배송지 변경, 오배송, 파손 배송 문의
