# ALF Escalation - live-test-mini-faq

## 즉시 이관 트리거

- 고객이 특정 주문의 배송 상태, 출고일, 도착 예정일을 묻는 경우
- 상품 하자, 오배송, 파손, 구성품 누락, 사용 흔적 또는 포장 훼손 판단이 필요한 경우
- 쿠폰 만료일, 복구 여부, 부분 취소, 이벤트 쿠폰 예외처럼 계정별 확인이 필요한 경우
- 환불 금액, 환불 처리 기간, 결제 취소 상태처럼 제공 FAQ에 없는 결제 관련 질문

## 추가 질문 후 이관

- 교환/반품 문의: 상품 수령일과 단순 변심인지 여부를 확인한 뒤, 7일 초과 또는 상태 판단이 필요하면 이관한다.
- 쿠폰 문의: 주문 취소 여부와 쿠폰 유효기간 확인 가능 여부를 묻고, 계정 확인이 필요하면 이관한다.
- 배송 문의: 일반 기간은 안내하되, 고객이 주문별 조회를 원하면 이관한다.

## 이관 제외 조건

- 고객이 일반 배송 기간만 묻는 경우: 결제 완료 후 영업일 기준 2~3일 안내
- 단순 변심 교환/반품 신청 기한만 묻는 경우: 상품 수령 후 7일 이내 안내
- 만료 쿠폰 복구 여부만 묻는 경우: 만료된 쿠폰은 복구되지 않음 안내

## 이관 문구

- "쿠폰 유효기간과 취소 처리 상태는 계정별 확인이 필요해 상담원에게 연결해 드리겠습니다."
