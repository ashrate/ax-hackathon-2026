# 바잇미 ALF 지식 범위

## ALF 답변 범위

바잇미 고객이 공개 FAQ, 이용안내, 상품 상세 정책에서 확인할 수 있는 주문, 배송, 교환, 반품, 환불, 쿠폰, 적립금, 회원 정보 질문에 답한다.

채널톡 공식 사례에 따르면 바잇미는 배송, 교환, 반품, 주문 취소처럼 반복되는 문의를 FAQ와 ALF로 처리해 상담사가 중요한 상담에 집중하도록 운영했다. 이 지식 묶음은 그 공개 사례와 바잇미 공개 페이지를 기준으로 만든 ALF 초안이다.

## 확정 지식

- 바잇미는 공개 FAQ에서 결제/배송, 교환/반품/환불, 회원가입/정보, 마일리지적립, 기타 카테고리의 질문을 제공한다.
- 공개 FAQ에는 비회원 주문조회, 쿠폰 사용, 묶음배송, 배송 휴무, 주문 취소, 교환/반품 불가 조건, 카드 환불 소요, 오배송/누락 처리 같은 반복 문의가 포함되어 있다.
- 이용안내는 회원가입, 적립금, 상품 주문 절차, 주문 확인, 배송기간, 주문취소/교환/환불의 기본 정책을 제공한다.
- 상품 상세 페이지에는 공급사 직접 배송/교환 여부, 택배사, 배송비, 무료배송 기준, 반품 가능 기간, 교환/반품 배송비가 상품별로 다르게 표시된다.

## 답변 시 주의

- FAQ와 상품 상세 정책이 모두 있는 경우 상품 상세 정책을 우선 확인하도록 안내한다.
- 고객 주문 상태, 송장 번호, 실제 환불 완료 여부는 공개 문서만으로 확인할 수 없으므로 ALF가 추정하지 않는다.
- 상품별 반품 가능 기간이 2일 또는 7일로 다르게 나타나는 예시가 있으므로 공통 반품 기간으로 단정하지 않는다.

## 상담원 확인 필요

- 고객의 실제 주문 상태가 `상품준비중`인지 `결제완료`인지 확인해야 하는 주문 취소/변경 요청.
- 특정 상품의 상세 페이지에 없는 반품 가능 기간, 제주/도서산간 추가비, 제작상품 여부.
- 상품 하자 판정, 오배송 사진 확인, 누락 상품 재발송 또는 환불 처리.
