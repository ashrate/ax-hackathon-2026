# 알라딘 공개 출처 범위

## ALF 답변 범위

알라딘 공개 고객센터 FAQ와 로그인 없이 접근 가능한 FAQ AJAX 응답을 바탕으로 배송/수령, 반품/교환, 고객센터 연결 기준을 안내한다.

## 확정 지식

- 고객센터 FAQ에는 배송/수령일안내와 반품/교환 카테고리가 공개되어 있다.
- 고객센터 운영시간 영역에는 전화번호 `1544-2514`, 평일 `09:00 ~ 18:00`, 토요일과 공휴일 휴무가 표시되어 있다.
- FAQ 상세 답변은 공개 AJAX URL로 로딩되며, 라이트 실행에서는 반품 가능 여부, 반품 신청, 지정택배 방문, 하자 교환, 편의점 픽업, 배송일 지정, 출고 지연 관련 응답을 확인했다.

## 답변 시 주의

- 주문별 배송 상태, 환불 완료 여부, 고객 계정의 주문 내역은 공개 FAQ만으로 조회한 것처럼 말하지 않는다.
- 전자책, 전자제품류, 중고매장 상품, 회원간 직거래 상품은 일반 도서와 조건이 다르므로 공통 정책으로 합치지 않는다.
- 고객센터 블로그 글은 공개 근거로 사용할 수 있지만, 최신 FAQ AJAX 응답과 함께 검토하고 충돌 가능성은 운영자 확인 대상으로 남긴다.

## 상담원 확인 필요

- 특정 주문의 현재 처리 단계.
- 고객 계정별 환불 완료 여부.
- 상품별 반품 불가 표시와 판매자직배송 상품의 실제 협의 경로.
