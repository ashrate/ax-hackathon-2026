# 교환 반품 환불 FAQ

## ALF 답변 범위

교환 접수, 오배송/누락, 불량 상품, 반품 배송비, 카드 환불 지연, 주문 취소를 안내한다.

## 확정 지식

- 사이즈가 맞지 않는 경우 FAQ는 마이페이지에서 직접 교환 접수할 수 있다고 안내한다. 교환 접수 후 배송비 결제가 완료되면 업체가 상품을 수거하고 새 사이즈 상품을 발송한다.
- 오배송이 발생하면 전체 박스 사진, 외부 송장 사진, 내부 수령 상품 전체 사진을 준비해 바잇미 고객센터 또는 채팅상담으로 연락하라고 안내한다.
- 상품 불량은 표시/광고 내용 또는 계약 내용과 다른 경우 상품을 받은 날부터 3개월 이내, 그 사실을 알게 된 날부터 30일 이내에 교환 및 환불 요청이 가능하다고 FAQ에 안내되어 있다.
- 상품 상태 사진을 첨부하면 더 빠른 상담이 가능하다고 FAQ에 안내되어 있다.
- 오배송/누락은 수령 후 영업일 기준 5일 이내에 상품을 확인하고 연락해야 원활한 처리가 가능하다고 FAQ에 안내되어 있다.
- 배송완료 후 단순변심 반품 신청은 FAQ 기준 7일 이내 가능하다고 안내되어 있다. 단, 상품별 상세 정책이 다를 수 있다.
- 단순변심 반품은 고객 부담이며, 초기 무료배송으로 받은 경우 초기 배송비까지 지불해야 환불이 가능하다고 FAQ에 안내되어 있다.
- 카드환불은 카드사 사정에 따라 환불 접수일로부터 영업일 기준 3~7일 정도 소요될 수 있다고 FAQ에 안내되어 있다.
- 결제완료 단계의 주문 취소는 마이페이지의 주문/배송, 주문목록, 주문상세페이지에서 직접 가능하다고 FAQ에 안내되어 있다.
- 상품준비중 이후 단계에서는 주문 취소가 제한된다고 FAQ에 안내되어 있다.

## 답변 시 주의

- 상품 상세 페이지마다 단순변심 반품/교환 가능 기간이 다르므로 반드시 상품명을 확인한다.
- 상품 하자 여부는 ALF가 판정하지 않고 사진 접수와 상담원 확인으로 안내한다.
- 카드 환불 지연은 카드사 사정이 포함될 수 있으므로 7영업일 이후 미처리 시 상담원 연결로 안내한다.

## 상담원 확인 필요

- 오배송, 누락, 불량, 파손, 사용 흔적 여부 확인.
- 고객이 사진을 제출한 뒤 재발송 또는 환불 처리 여부.
- 상품준비중 이후 주문 취소 가능성.
- 결제수단별 실제 환불 완료 여부.
